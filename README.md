# Gerenciador de Loterias (SPA)

Uma Aplicação Web de Arquivo Único (Single Page Application) para gerenciamento e conferência de bilhetes de loterias da CAIXA.

![Preview](preview.png)
*(Você pode adicionar um print aqui depois)*

## Funcionalidades

*   **Arquivo Único:** Todo o sistema (HTML, CSS, JS) roda em um único arquivo `index.html`.
*   **Modalidades Suportadas:** Mega-Sena, Lotofácil, Quina, Lotomania, Timemania e Dia de Sorte.
*   **Regras de Negócio:** Validação de quantidade mínima de dezenas e ranges de números.
*   **Conferência em Tempo Real:**
    *   Insira os números sorteados e veja os bilhetes "acenderem" instantaneamente.
    *   Ordenação automática: bilhetes com mais acertos sobem para o topo.
*   **Persistência:** Os dados são salvos no navegador (LocalStorage), então você não perde seus jogos ao fechar a página.
*   **Dark Mode:** Interface moderna e agradável.

## Como Usar

1.  Baixe o arquivo `index.html`.
2.  Abra em qualquer navegador moderno (Chrome, Firefox, Edge).
3.  Cadastre seus jogos e divirta-se!

## Tecnologias

*   HTML5
*   CSS3 (Modern Flexbox/Grid)
*   JavaScript (Vanilla)
